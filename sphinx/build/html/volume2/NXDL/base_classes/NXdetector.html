Template of a detector, detector bank, or multidetector. The indices require explanation:

np

the number of points in a scan. This dimension is only present in scanning measurements.

tof

the number of points in a scan. This dimension is only present in scanning measurements.

i

the number of pixels in the slowest varying direction. This is only missing in the point detector.

j

the number of pixels in the fastest varying direction. This exists only in “area” detectors.
